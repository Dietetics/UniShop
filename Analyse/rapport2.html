Tâche 1 : Analyse

Plusieurs parties de notre système UniShop seront informatisées, telles que les interfaces utilisateur qui seront développées pour permettre aux acheteurs et aux revendeurs de s'inscrire, de se connecter et de gérer leurs profils. Des formulaires interactifs seront mis en place pour recueillir les informations nécessaires. Les données seront stockées dans une base de données sécurisée utilisant PHP et SQL. Aucun matériel spécifique n'est requis, car le système peut fonctionner sur des serveurs cloud. Les interactions avec d'autres systèmes se feront via des API. Les utilisateurs interagiront avec le système via une interface utilisateur conviviale, comprenant des menus intuitifs et une navigation moderne.
